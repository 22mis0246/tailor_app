## Customer Features

- **Browse Designs** — Scrollable feed of handcrafted outfits with real photos, ratings and pricing
- **Filter by Category** — Instantly filter by Kurta · Blouse · Shirt · Salwar · Sherwani
- **Design Detail Page** — Full description, tailor name, star rating, delivery time and fabric guarantee
- **Measurement Form** — Enter chest, waist, hips, shoulder and length in a clean two-column layout
- **Place Order** — One-tap order confirmation with animated bottom sheet success state
- **My Orders** — View all placed orders with status badge and full measurement breakdown

## Tailor Features

- **Dashboard** — At-a-glance stats: total designs uploaded, orders received and rating
- **My Designs Tab** — View all published designs with thumbnail, category tag and price
- **Orders Tab** — Expandable order cards showing customer name, measurements and special instructions
- **Add Design** — Publish a new design with title, category, price and description
- **Photo Upload** — Image upload placeholder ready for Firebase Storage integration
